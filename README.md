# Royal Comfort - Платформа премиум-благоустройства 🏡

Полноценное веб-приложение для заказа беседок, бань, купелей и малых архитектурных форм премиум-класса с интерактивным конфигуратором и интеграцией с Telegram-ботом. 

Проект разработан как дипломная работа и готовое коммерческое решение для микробизнеса в сфере премиум-услуг.

## 🌟 Ключевые возможности

### Для клиентов (Сайт)
- 💅 **Премиальный UI/UX:** Современный адаптивный дизайн с анимациями, темной темой и плавными переходами (Glassmorphism, Tailwind CSS, Framer Motion).
- 🛠️ **Интерактивный конфигуратор:** Пошаговая сборка изделия (беседки, чаны и др.) с выбором опций, автоматическим расчетом итоговой цены и применением скидок.
- 📱 **Мобильная адаптивность:** Полная поддержка всех устройств (от смартфонов до 4K мониторов).
- 🎁 **Промо и Реферальная система:** Возможность ввода реферального кода, получение подарков и скидок.
- 💬 **Мгновенная связь:** Глобальный Telegram-виджет и удобные формы для быстрой связи с менеджером.

### Для владельца бизнеса (Telegram-бот)
- 🛒 **CRM в кармане:** Получение уведомлений о новых заказах и консультациях с полным списком выбранных конфигураций, подарков и комментариев.
- 📊 **Управление статусами:** Перевод заказов по воронке продаж (Обработка ➔ Утверждение ➔ Производство ➔ Установка ➔ Завершен).
- 🛍️ **Управление каталогом (CMS):** Добавление, редактирование и скрытие товаров, категорий, опций и цен прямо с телефона без навыков программирования.
- ⭐ **Модерация отзывов:** Проверка и публикация отзывов клиентов на сайте через бота.
- 💾 **Безопасность и Бэкапы:** Защита от угона аккаунта по ID, возможность создания и восстановления резервных копий БД одним нажатием.

## 🛠 Технологический стек

- **Фронтенд:** React 18, Vite, Tailwind CSS, Framer Motion, React Router, Lucide React, Swiper.
- **Бэкенд:** Node.js, Express, Sequelize (ORM), SQLite (База данных).
- **Интеграция:** Telegraf (Telegram Bot API).
- **Безопасность:** express-rate-limit, CORS, защита от Path Traversal, строгая валидация Telegram ID.

## 🚀 Установка и запуск

Проект спроектирован для работы в режиме монолита на одном сервере, где Express раздает как API, так и скомпилированные файлы React-приложения.

### Предварительные требования
- Node.js (v18+)
- NPM или Yarn

### 1. Подготовка окружения
Клонируйте репозиторий и перейдите в папку сервера:
```bash
git clone <url-репозитория>
cd royal-comfort-fullstack/royal-comfort-server
```

Установите зависимости сервера:
```bash
npm install
```

Перейдите в папку клиента и установите его зависимости:
```bash
cd ../royal-comfort-client
npm install
```

### 2. Сборка клиентской части
Соберите React-приложение:
```bash
npm run build
```
*(Скомпилированные файлы появятся в папке `royal-comfort-client/dist`)*

### 3. Настройка конфигурации (.env)
В папке `royal-comfort-server` создайте файл `.env`:
```env
PORT=5000
BOT_TOKEN=ваш_токен_telegram_бота
```

### 4. Первичный запуск и заполнение БД (опционально)
Если вы запускаете проект впервые, вы можете наполнить базу данных демо-контентом:
```bash
npm run seed
```

### 5. Запуск сервера
Перейдите в папку `royal-comfort-server` и запустите приложение:
```bash
npm start
```
Сервер будет доступен по адресу: `http://localhost:5000`

## 🛡 Безопасность
Все маршруты изменения данных закрыты от внешнего API. Управление каталогом, заказами и контентом осуществляется исключительно через Telegram-бота, что исключает возможность несанкционированного доступа к базе данных со стороны сайта. Доступ к админ-панели бота жестко привязан к неизменяемым `Telegram ID`.
