# 🏰 Royal Comfort — Full-Stack Платформа (Дипломный проект)

> Премиальный интернет-магазин и CMS для производителя банных чанов, беседок и уличных объектов отдыха.
> Разработано с использованием **React + Node.js + SQLite + Telegram Bot API**.

## 📌 О проекте

**Royal Comfort** — это production-ready веб-платформа с инновационной системой управления контентом. Вместо классической админ-панели (которую нужно долго разрабатывать и неудобно использовать с телефона), проект предлагает **Telegram-бота**, который выступает в роли полноценной CMS и CRM-системы.

**Роли пользователей:**
- **Клиенты:** просмотр каталога, конфигурация товаров в реальном времени, отслеживание статуса заказа по номеру, чтение и публикация отзывов.
- **Администраторы/Менеджеры:** полное управление контентом сайта (товары, категории, акции, отзывы, заказы) прямо через интерфейс Telegram.

---

## ✨ Ключевой функционал

### 🌐 Клиентская часть (Frontend)
- **Современный UI/UX:** Дизайн с использованием Glassmorphism, плавных анимаций (Framer Motion) и адаптивной верстки (TailwindCSS).
- **Интерактивный конфигуратор:** Пошаговый выбор параметров товара (размер, материал, цвет, дополнительные опции) с мгновенным перерасчетом итоговой стоимости.
- **Трекинг заказов:** Специальная страница `/tracking`, где клиент может по ID заказа узнать текущий этап производства (от "Обработки" до "Готовности к выдаче").
- **Система отзывов:** Интегрированная лента отзывов с модерацией. Клиент оставляет отзыв через сайт или бота, после чего он уходит на проверку.

### 🤖 Бэкенд и Telegram-бот (Backend & CMS)
- **Telegram CMS:** Бот позволяет добавлять новые категории, скрывать товары, менять цены, назначать скидки и изменять обложки товаров. Сайт обновляется мгновенно.
- **Обработка заказов (CRM):** Бот уведомляет менеджеров о новых заявках. В два клика можно сменить статус заказа, перевести "Консультацию" в полноценный "Заказ" и уведомить клиента.
- **Управление медиа:** Загрузка фотографий для товаров и отзывов происходит прямо в чат с ботом. Сервер автоматически скачивает изображения и раздает их фронтенду.
- **Отказоустойчивость:** Валидация всех текстовых и числовых вводов, изоляция состояний через "Сцены" (Telegraf Scenes), защита от несанкционированного доступа (ролевая модель).

## 🏗️ Технический стек

| Слой | Технологии |
|------|-----------|
| **Frontend** | React 18, Vite, TailwindCSS, Framer Motion, React Router v6, Swiper, Lucide React |
| **Backend** | Node.js, Express.js, Sequelize ORM |
| **База данных** | SQLite (встроенная, через `better-sqlite3`, с возможностью миграции на PostgreSQL) |
| **Telegram Bot** | Telegraf.js (Сцены, Inline-клавиатуры, Session state) |
| **Деплой** | Подготовлено для размещения на VPS (PM2, Nginx) |

## 🗂️ Архитектура

```text
royal-comfort-fullstack/
├── royal-comfort-client/        # React Фронтенд
│   ├── public/                  # Статика (видео, заглушки)
│   ├── src/
│   │   ├── components/          # Модальные окна, карточки, layout
│   │   ├── context/             # Global State (каталог, активные подарки)
│   │   └── pages/               # Home, Catalog, Tracking
│   └── vite.config.js           # Проксирование API и настроек сборки
│
└── royal-comfort-server/        # Node.js Бэкенд
    ├── src/
    │   ├── controllers/         # Логика REST API (заказы, отзывы)
    │   ├── models/              # Структуры таблиц базы данных (Sequelize)
    │   ├── routes/              # Маршрутизация HTTP-запросов
    │   ├── services/
    │   │   └── bot/             # Telegram CMS (Бот)
    │   │       ├── handlers/    # Обработка инлайн-кнопок
    │   │       ├── scenes/      # Изолированные многошаговые сценарии
    │   │       └── keyboards/   # Клавиатуры интерфейса бота
    │   └── utils/               # Загрузчик файлов (fileDownloader)
    └── server.js                # Точка входа
```

## 🚀 Быстрый старт (Для разработки)

### 1. Клонирование и установка
```bash
git clone https://github.com/Makstein666/Royal_Comfort.git
cd Royal_Comfort

# Зависимости сервера
cd royal-comfort-server && npm install

# Зависимости клиента
cd ../royal-comfort-client && npm install
```

### 2. Настройка окружения
В папке `royal-comfort-server` создайте файл `.env`:
```env
PORT=5000
BOT_TOKEN=ваш_токен_telegram_бота
```

### 3. Запуск
```bash
# Из корня проекта:
npm run start:all
```
*Фронтенд запустится на порту `5173`, Бэкенд на `5000`.*
