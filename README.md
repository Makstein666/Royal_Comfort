# Royal Comfort

Сеть производства товаров для премиального дачного отдыха под "ключ".

Данный репозиторий содержит исходный код клиентской и серверной частей проекта **Royal Comfort**. Приложение включает в себя интерактивный конфигуратор продукции для клиентов, панель администратора на базе Telegram-бота и полноценный бэкенд для обработки заказов.

## 🚀 Функциональные возможности

### Для клиентов (Frontend)
- **Современный UI/UX:** Адаптивный и привлекательный дизайн, разработанный с использованием React и Tailwind CSS.
- **Интерактивный конфигуратор:** Позволяет пользователям собирать уникальный продукт, выбирая опции и мгновенно видя изменение стоимости.
- **Отслеживание заказа:** Проверка статуса производства заказа в реальном времени.

### Для администраторов (Telegram-бот)
- **Управление заказами:** Просмотр, изменение статусов и отслеживание статистики прямо с телефона.
- **Управление каталогом:** Добавление/скрытие категорий и изменение цен без необходимости заходить в админку сайта.
- **Уведомления:** Мгновенное получение информации о новых заказах и заявках на консультацию.

## 🛠 Технологический стек

### Client (Frontend)
- **Фреймворк:** React (сборщик Vite)
- **Стилизация:** Tailwind CSS
- **Анимации:** Framer Motion
- **Маршрутизация:** React Router DOM

### Server (Backend)
- **Среда выполнения:** Node.js
- **Фреймворк:** Express.js
- **База данных:** SQLite (через Sequelize ORM)
- **Интеграция:** Telegraf (Telegram Bot API)

## 💻 Установка и запуск (Локально)

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/Makstein666/Royal_Comfort.git
   ```

2. Установите зависимости для сервера:
   ```bash
   cd royal-comfort-server
   npm install
   ```

3. Установите зависимости для клиента:
   ```bash
   cd ../royal-comfort-client
   npm install
   ```

4. Создайте файл `.env` в папке `royal-comfort-server` и добавьте свой токен Telegram-бота:
   ```env
   BOT_TOKEN=ваш_токен_здесь
   PORT=5000
   ```

5. Запустите сервер:
   ```bash
   cd royal-comfort-server
   npm run start
   ```

6. В новом окне терминала запустите клиент:
   ```bash
   cd royal-comfort-client
   npm run dev
   ```
