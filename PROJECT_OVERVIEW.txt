================================================================================
                    ПРОЕКТ: ROYAL COMFORT (FULL-STACK)
================================================================================

1. ОБЩЕЕ ОПИСАНИЕ
-----------------
Royal Comfort — это премиальная платформа для производства и продажи товаров для 
дачного отдыха (банные чаны и т.д.). Система включает современный клиентский 
сайт с конфигуратором и административную панель, реализованную на базе 
Telegram-бота.

2. ТЕКУЩАЯ АРХИТЕКТУРА
----------------------
- Frontend: React + Tailwind CSS + Framer Motion (сборка Vite).
- Backend: Node.js + Express + Sequelize (ORM).
- База данных: SQLite (один файл database.sqlite, легко переносится).
- CMS / Админка: Telegram Bot (библиотека Telegraf).

3. ОСНОВНЫЕ ВОЗМОЖНОСТИ
-----------------------
- Интерактивный конфигуратор: расчет стоимости в реальном времени.
- Умный каталог: возможность скрывать категории и переводить их в режим предзаказа.
- Система уведомлений: моментальные сообщения админам о новых заявках в Telegram.
- CMS в боте: управление товарами, ценами и опциями прямо из чата.
- Акции и подарки: динамическая настройка подарков для разных категорий через бота.

4. ЧТО БЫЛО СДЕЛАНО (ПОСЛЕДНИЕ ИЗМЕНЕНИЯ)
-----------------------------------------
- Рефакторинг бота: Переход от монолитного файла к модульной архитектуре 
  (handlers, scenes, keyboards).
- CMS-функционал: Добавлены сцены пошагового создания товаров и опций в боте.
- Система бэкапов: Реализована возможность создания резервных копий базы данных 
  и отката к ним через интерфейс бота.
- Управление админами: Реализована динамическая авторизация через /admin_login.
- Безопасность: Внедрена защита от спама (Rate Limiting) на API запросы.
- Исправление ошибок: Устранены проблемы с синхронизацией SQLite и инициализацией 
  бота без токена.

5. ИНСТРУКЦИЯ ПО ЗАПУСКУ
------------------------
1. В папке royal-comfort-server создать .env с BOT_TOKEN и ADMIN_PASSWORD.
2. Выполнить npm install в папках client и server.
3. Запустить сервер: cd royal-comfort-server && npm start.
4. Запустить клиент: cd royal-comfort-client && npm run dev.
